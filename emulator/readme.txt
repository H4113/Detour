Un petit emulateur en NodeJS pour le client.

Modifiez le fichier config.js pour mettre l'adresse ip du serveur 
ainsi que le bon port.

Usage:
	node emulator.js 
					lance 1 client, voir le résultat avec index.html
	node emulator.js <nombre de clients à lancer>
	node emulator.js clean 
					pour nettoyer le fichier data/benchmark.json
	
	node analog.js
	node analog.js -d
					pour obtenir des informations de benchmark 
					voir le résultat avec graph.html
